=======================
django-email-foundation
=======================

**django-email-foundation** is a Django package for help you to create emails using the "zurb foundation" technologies.
Take a look to `Foundation for Emails`_ documentation for more details.

.. _`Foundation for Emails`: https://foundation.zurb.com/emails/docs/

It provide you the following commands:

* A django command for install the required node packages, such *gulp* and *inky*.
* A django command for create basic tree directory with the essential email layouts.
* A django command for watch and build your email's templates and store to the target folder.
